# PetLog Mobile Web App

반려동물의 사진, 기록, 일정, 병원 정보를 모바일 사용 흐름에 맞춰 관리할 수 있도록 설계한 웹앱 프로젝트입니다.  
앱형 화면 전환, 기록 관리, 사진 아카이브, 병원 정보 탐색 흐름을 HTML, CSS, JavaScript로 구현했습니다.

## Live Demo

https://chlghdwns960-wq.github.io/petlog-mobile-webapp/

## Role

- 모바일 웹앱 화면 구조 설계
- Splash, Onboarding, Home, Record, Photo, Hospital 화면 구현
- JavaScript 기반 화면 전환 및 인터랙션 구성
- 반려동물 기록 관리 흐름 정리
- GitHub Pages 배포

## Tech

- HTML5
- CSS3
- Vanilla JavaScript
- AI-assisted workflow

## Screen Flow

`Splash` → `Onboarding` → `Login` → `Home` → `Record` → `Photo` → `Schedule` → `Hospital` → `My Page`

## Key Points

- 모바일 화면 기준의 정보 구조 설계
- 반려동물 관리 흐름을 기능별 화면으로 분리
- 카드, 탭, 리스트 등 반복 UI를 활용한 화면 구성
- 사용자가 기록을 확인하고 다음 행동으로 이동하기 쉬운 구조 설계

## Main Files

- `index.html` - Splash
- `index2.html` - Onboarding
- `index5.html` - Home / Record summary
- `index6.html` - Photo archive
- `index7.html` - Record detail
- `index9.html` - Schedule list

- `script/`
