# PetLog Mobile Web App

반려동물의 사진, 기록, 일정, 병원 정보를 모바일 흐름에 맞춰 관리할 수 있도록 설계한 웹앱 프로젝트입니다.

## Live Demo

https://chlghdwns960-wq.github.io/petlog-mobile-webapp/

## Original Demo

https://chlghdwns960-wq.github.io/ai/Exam

## Role

- UI 구조 설계
- 모바일 화면 구현
- JavaScript 인터랙션 구성
- 기록, 사진, 병원 정보 탐색 흐름 정리

## Tech

- HTML5
- CSS3
- Vanilla JavaScript
- LocalStorage

## Main Files

- `index.html`
- `index2.html` - 온보딩
- `index5.html` - 홈/기록 요약
- `index6.html` - 사진 아카이브
- `index7.html` - 기록 상세
- `index9.html` - 일정 목록
- `index11.html` - 병원 리스트
- `index12.html` - 병원 상세
- `index13.html` - 병원 지도
- `css/`
- `script/`
- `images/`

## Screen Flow

`Splash` → `Onboarding` → `Login` → `Home` → `Record / Photo / Schedule / Hospital / My Page`

## Portfolio Note

기록 추가, 사진 확인, 병원 정보 탐색 흐름을 모바일 사용 순서에 맞춰 단순화하는 데 초점을 둔 프로젝트입니다.
